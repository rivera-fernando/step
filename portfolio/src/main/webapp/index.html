Personal Projects

Swampy Sublease
(Fall 2020 - Present)

This project began during SwampHacks 2020, but my team and I decided to continue improving upon it. It aims to centralize and solve the problem of advertising off-campus apartment subleases for University of Florida students. It utilizes Python & Django for backend, MongoDB for storage, SendGrid for email services, MaterializeCSS for frontend, and Heroku for deployment.

Transcript Formatter
(Fall 2020)

After downloading my unofficial transcript from the UF website countless times, I realized it was less than ideal when converted to a PDF. I sought to create a version that would format it in a better way, creating a more informative experience. It utilizes Python & Django for backend, MaterializeCSS for frontend, Selenium for scraping, and Heroku for deployment.

Paracosm File Uploader
Fall 2019

Class project for CEN3031. My team and I created a tool for Paracosm, a local Gainesville company to have access to a versioned history of their device-unique calibration files. I served as the scrum master for a 5 person team, and took part in daily standup meetings, sprint retrospectives as well as other agile techniques. It utilizes JavaScript & Node with Express for backend, React for frontend, and S3 bucket for storage, and Firebase for authentication.
